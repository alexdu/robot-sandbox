robot-sandbox

Robot arm simulator with rigid body dynamics
Based on V+ language

INSTALL:

Binary release (Windows): unzip and run :)

Windows source:

- Python 2.5
- ipython
- cgkit
- ODE (0.10+ preferred)
- pyode (from SVN)
- numpy
- pygame
- PyProtocols (from source, pyhon setup.py --without-speedups install)
- pyreadline
