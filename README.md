Demos
-----

Hanoi towers:

    env hanoi
    load hanoi
    exec hanoi_main

Conveyor belt:

    env belt
    load belt
    exec belt
    
Robot drawing (using a ballpoint pen and paper):

    env desen
    load desen
    exec rose(3,2)
