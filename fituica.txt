
Programele se afla in "robot-programs"

Cateva comenzi posibile:

mc                     # monitor-commands (un fel de help)
env hanoi              # initializeaza mediul de lucru
load hanoi             # incarca un program robot (din hanoi.v2)
exec hanoi_main        # ruleaza un program robot
speed 50               # seteaza viteza monitor
status                 # afiseaza vitezele si programul care ruleaza
see hanoi_main         # editeaza un program robot
see hanoi.v2           # editeaza un fisier cu programe robot

Taste scurtatura in consola: TAB, sageata sus.
