robot-sandbox by alexdu

Robot arm simulator with rigid body dynamics

Dependencies

Windows: all dependencies are included in the binary distribution. Linux: python 2.6, ipython, cgkit, pyode, pyopengl, numpy, cellulose... and maybe others

Install

Windows: unzip and run Linux: you have to use some console black magic...
